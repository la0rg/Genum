Добро пожаловать на страницу проекта Genum.

Genum - расширение для популярной программы Anki. Anki помогает в запоминании иностранных слов, используя современные методики обучения, тем самым уменьшая время на запоминание. Если прежде вы не пользовались данной программой желательно изучить статью, однако чтобы начать пользоваться разработанным расширением, будет достаточно данного руководства.

Genum и автоматическая генерация карточек.

Одной из полезных возможностей Anki - является добавление слов (карточек) вручную, вы можете добавить лексику из книги, сериала или слова для предстоящего теста. Однако, формирование карточки является довольно рутинным занятием, для автоматизации данного действия был разработан Genum.

Основные возможности Genum:

  • Создание карточек для англо-русских колод (EN-RU)
  • Автоматическая генерация:
    • Перевод
    • Примеры (контекст) с переводом
    • Определения на языке оригинала
    • Произношение на языке оригинала
    • Изображение
    • Транскрипция
  • Локальное сохранение медиа-контента (карточку можно просматривать без подключения к интернету)

Установка и настройка

Чтобы начать использовать Anki с расширение Genum следуйте инструкции:

  1. Установите Anki
  2. Установите расширение
    Для этого перейдите в пункт меню: Инструменты -> Дополнения -> Обзор и установка ...
    В появившемся окне, введите код: 237340276 и нажмите ОК.
    Чтобы изменения вступили в силу, перезапустите Anki.
  3. Добавьте новый тип записей
    Инструменты -> Create Genum card template...

Использование Genum

Для использования плагина откройте окно редактора новой карточки (Кнопка "Добавить" в главном окне Anki)

  • Убедитесь, что выбран тип карты Genum и верная колода.
  • Введите английское слово в поле word.
  • Нажмите на кнопку с лампочкой - "Genum start" (Ctrl+G).
  • После того как все поля будут заполнены, нажмите кнопку "Добавить".
На этом создание карточки завершено. Можете приступать к изучению.

Конфигурация расширения

Более опытные пользователи могут настроить порядок полей, количество примеров и определений, а также собственные API ключи.
Для этого перейдите в меню: Инструменты -> Дополнения -> Genum -> Редактировать, откроется редактор конфигурации. После редактирования данного файла, закройте редактор и перезагрузите Anki.

Распространение и поддержка

Программа распространяется под лицензией MIT.
ДАННАЯ ВЕРСИЯ ПРОГРАММЫ ЯВЛЯЕТСЯ ТЕСТОВОЙ
Для работы расширения используются сторонние сервисы, на которые получены пробные ключи, с ограничением на количество запросов в месяц. Если данные лимиты будут исчерпаны, то расширение не сможет работать корректно.
Для получения собственных ключей для данных сервисов, пройдите по ссылкам:

  1. collinsdictionary
  2. bingsearchapi
  3. Yandex dictionary
  4. Yandex translate
Для установки полученных ключей обратитесь к разделу "Конфигурация расширения".
