Добро пожаловать на страницу проекта Genum.

Genum - расширение для популярной программы Anki. Anki помогает в запоминании различных вещей (в т.ч. иностранных слов), используя современные методики обучения, тем самым уменьшая время на запоминание. Если прежде вы не пользовались данной программой желательно изучить статью, однако чтобы начать пользоваться разработанным расширением, будет достаточно данной страницы.

Genum или автоматическая генерация карточек.

Поупражнявшись с возможностями Anki и изучив несолько заготовленных колод с сайта, вы непременно захотите добавлять и изучать те слова, которые интересны именно вам. Будь то лексика книги, сериала или подготовка к предстоящему тесту. Однако, добавлять каждое слово в ручную довольно рутинное занятие (особенно если вы хотите видеть не только перевод). Для решения данной проблемы и был разработан Genum.

Основные возможности Genum:

  • Создание карточек для англо-русских колод (EN-RU)
  • Автоматическая генерация:
    • Перевод
    • Примеры (контекст) с переводом
    • Определения на языке оригинала
    • Американсокое произношение слова
    • Изображение
    • Транскрипция
  • Локальное сохранение медиа-контента (карточку можно просматривать без подключения к интернету)

Установка и настройка

Чтобы начать использовать Anki с расширение Genum следуйте инструкции:

  1. Установите Anki
  2. Добавьте новый тип записей
    3. Инструкция
    1. Перейдите в меню: Инструменты -> Управление типами записей...
    2. Нажмите кнопку "Добавить", выберите "Добавить: Основная" и нажмите ОК
    3. Введите название (напр. Genum), нажмите ОК
    4. Выберите новый тип записи и нажмите кнопку "Поля..."
    5. Создайте и переименуйте поля, как показано на рисунке. Затем нажмите Закрыть.
    6. Выберите новый тип записи и нажмите кнопку "Карточки..."
    7. Создаем шаблон карточек. Для этого заполняем три левых блока кодами, приведенными в таблице:
      Шаблон лицевой стороны (вопроса) <div class=word>{{word}}</div> <div class=transc>[{{transcription}}]</div> [sound:{{sound}}] <div class=cont>{{context}}</div> <div class=cont>{{definition}}</div>
      Таблица стилей .card { font-family: Arial,sans-serif; text-align: center; background-color: #F5F2E7; } .word { font-size: 28px; color: #333333; } .transc { font-size: 16px; color: #317FC0; font-family: Arial Unicode MS; } .transl { font-size: 22px; color: #995B36; margin: 0 0 10px; } .cont { font-size: 15px; font-style: italic; color: #666666; margin: 10px 0 0; } .cont b { color: green !important; }
      Шаблон обратной стороны {{FrontSide}} <hr id=answer> <div class=transl>{{translation}}</div> <div class=cont>{{context_translation}}</div> <img src="{{image}}">
  3. Установите расширение

    4. Для этого перейдите в пункт меню: Инструменты -> Дополнения -> Обзор и установка ...
    В появившемся окне, введите код: 237340276 и нажмите ОК.
    Чтобы изменения вступили в силу, перезапустите Anki.

Использование и настройка Genum

Для использования плагина откройте окно редактора новой карточки (Кнопка "Добавить" в главном окне Anki)

  • Убедитесь, что выбран ранее созданный тип карты и верная колода.
  • Введите интересующее английское слово в поле word.
  • Нажмите кнопку "Generate document" (Ctrl+G).
  • После того как все поля будут заполнены, нажмите кнопку "Добавить".
На этом создание карточки завершено. Можете приступать к изучению.

Более опытные пользователи могут настроить заполнение и порядок полей.
Для этого перейдите в меню: Инструменты -> Дополнения -> Genum -> Редактировать. Откроется редактор конфигурации, в котором установлены порядковые номера полей (раздел Card template). После редактирования данного файла, закройте редактор и перезагрузите Anki.

Распространение и поддержка

Программа распространяется под лицензией MIT.
ДАННАЯ ВЕРСИЯ ПРОГРАММЫ ЯВЛЯЕТСЯ ТЕСТОВОЙ
Для работы расширения используются сторонние сервисы, на которые получены пробные ключи, с ограничением на количество запросов в месяц. Если данные лимиты будут исчерпаны, то расширение не сможет работать корректно.
Для получения собственных ключей для данных сервисов, пройдите по ссылкам:

  1. collinsdictionary
  2. bingsearchapi
  3. Yandex dictionary
  4. Yandex translate
Для установки полученных ключей обратитесь к разделу "Использование и настройка Genum".
